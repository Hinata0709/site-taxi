# ID Taxi Website

Site vitrine statique pour le service de taxi ID Taxi à Ermont (95).

## Technologies
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (Vanilla)

## Structure
- `index.html`: Structure principale
- `styles.css`: Styles et animations
- `script.js`: Logique interactive (menu, scroll, animations)

## Déploiement
Le site est déployé automatiquement sur Vercel à chaque push sur la branche main.
